Dispatches the given event and optionally narrowing down the entities by the set of entity identifiers to dispatch the event to.

If no particular IDs are specified, the event will be dispatched according to the repository routing rules.

Return

the set of identifiers to which the event was actually dispatched

Parameters

repo

the projection repository which manages the entities involved into the catch-up

event

event to dispatch

narrowDownToIds

optional set of identifiers of the targets to narrow down the event targets
