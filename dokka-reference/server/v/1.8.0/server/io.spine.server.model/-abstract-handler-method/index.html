AbstractHandlerMethod

public abstract class AbstractHandlerMethod<T, M extends Message, C extends MessageClass<M>, E extends MessageEnvelope<? extends Object, ? extends Signal<?, ?, ?>, ? extends Object>, R extends MessageClass<?>> implements HandlerMethod<T, C, E, R>
Content copied to clipboard

An abstract base for wrappers over methods handling messages.

Two message handlers are equivalent when they refer to the same method on the same object (not class).

Parameters

<T>

the type of the target object

<M>

the type of the message handled by this method

<C>

the type of the message class

<E>

the type of message envelopes, in which the messages to handle are wrapped

<R>

the type of the produced message classes
