RecordBasedRepository

public abstract class RecordBasedRepository<I, E extends Entity<I, S>, S extends EntityState> extends Repository<I, E>
Content copied to clipboard

The base class for repositories that store entities as records.

Such a repository is backed by RecordStorage. Entity states are stored as EntityRecords.

Parameters

<I>

the type of entity identifiers

<E>

the type of stored entities

<S>

the type of entity state messages
