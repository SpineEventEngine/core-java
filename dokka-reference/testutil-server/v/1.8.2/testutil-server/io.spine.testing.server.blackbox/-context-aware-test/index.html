Abstract base for test suites based on BlackBoxContext.

Such a test suite creates a new instance of BoundedContext using a builder which implementing classes must provide. The context under the test is created before each test method and closed after each test method automatically.

Testing methods obtain the context for:

  1. Sending signals under the test to the context via receivesXxx() methods, such as receivesCommand().
  2. Asserting results of handling the signals via assertXxx() methods, e.g. assertEvents().

Test suites derived from this class must NOT close the context directly. Doing so will fail the test.
