A set of assertions for a Query execution result.

The class base methods check a set of entity states received in the QueryResponse. The subject can be used as follows: 

context.assertQueryResult(query)
       .containsExactly(state1, state2);
Content copied to clipboard

If the actual queried messages are required, the subject can return them as follows: 

context.assertQueryResult(query)
       .actual();
Content copied to clipboard

There are also convenience methods for checking response Status and received entity versions, as well as some others.

This class is not a "typical" Subject in a sense that it's not accessible to the outer world through it's factory, but is rather created with a custom staticmethod.
