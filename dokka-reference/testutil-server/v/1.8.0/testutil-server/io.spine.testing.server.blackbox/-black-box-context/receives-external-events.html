receivesExternalEvents

final BlackBoxContext receivesExternalEvents(    EventMessage first,     EventMessage second,     Array<EventMessage> other)
Content copied to clipboard

Sends off provided events to the Bounded Context as events from an external source.

The method accepts event messages or instances of io.spine.core.Event. If an instance of Event is passed, it will be posted to IntegrationBroker as is. Otherwise, an instance of Event will be generated basing on the passed event message and posted to the bus.

Return

current instance

Parameters

first

an external event to be dispatched first

second

an external event to be dispatched second

other

optional external events to be dispatched in the supplied order
