An annotation processor for the @Assign annotation.

Collects the types which contain command handler methods (marked with @Assign annotation) and writes them into the ${spineDirRoot}/.spine/spine_model.ser file, where "spineDirRoot" is the value of the spineDirRoot annotator option.

spineDirRoot is the only supported option of the processor. Use javac -AspineDirRoot=/path/to/project/root [...] to set the value of the option. If none is set, the option will default to current directory (denoted with "./").
